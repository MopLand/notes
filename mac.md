# Mac

## Brew

### 安装 Brew
	/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

### 使用 Brew
	brew install wget
	
## 更新 Git
	
### 查看 Git 版本
	git --version

### 安装 Git
	brew install git
	
### 更改 Git 指向
	brew link git --overwrite
	
### 查看 Git 版本
	git --version
	
## Mac 调优

### 安装非 App Store 应用

1. 系统偏好设置 > 安全性与隐私
	
2. 『点击锁按钮以进行更改』
	
3. 允许从以下位置下载的应用：任何来源

### 外接 USB 键盘
	
	# 系统偏好设置 > 键盘 > 修饰键
	
	# 选择键盘：[USB-KeyBoard]
	
	# 修改修饰键
		
	Control => Command
	Command => Control
